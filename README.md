# Agent Store — Claude Skills

Skills that help Claude use the **Agent Store** MCP connector (https://agent.store/mcp)
for Korean public & financial data.

## Skills in this repo
- **korean-public-data** (`korean-public-data/SKILL.md`) — guides Claude to answer
  Korean corporate, financial, legal, statistical, procurement, and real-estate
  questions using the right Agent Store data agent, with the correct identifiers
  and official source citations.

Connect: https://agent.store/mcp · Docs: https://agent.store/docs/connect/
